🌍 TripAdvisor Clone
A responsive front-end clone of the TripAdvisor website built with pure HTML and CSS.

📌 About the Project
This project is a static front-end clone of TripAdvisor, recreating its core UI including the navbar, hero search section, promotional banner, activity cards, and footer. Built as a practice project to strengthen HTML and CSS skills.

✨ Features

Responsive Navbar with logo, AI planning button, navigation links, currency selector, and sign-in button
Mobile-friendly — navbar collapses on small screens and shows a user icon instead
Hero Section with tabbed search bar (Hotels, Things to Do, Restaurants)
Promo Banner highlighting featured experiences
Interest Cards — Outdoors, Food, Culture, Water
Must-Do Experiences with ratings and pricing
Iconic Places section with destination images
Footer with links, site info, and copyright


🛠️ Built With

HTML5
CSS3 (Flexbox, Media Queries)
Google Fonts — Roboto
Font Awesome 7 — Icons
